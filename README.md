# 📖 [Rails with React] <a name="about-project"></a>


**[Rails with React]** is a project where I practice working with both `Rails` and `React` in a same repository. Here I have created an API end point that renders a data from the database.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

  <summary>Client</summary>
  <ul>
    <li><a href="https://reactjs.org/">React.js</a></li>
  </ul>
</details>

<details>
  <summary>Server</summary>
  <ul>
    <li><a href="https://rails.org/">Rails</a></li>
  </ul>
</details>

<details>

  <ul>
    <li><a href="https://www.postgresql.org/">PostgreSQL</a></li>
  </ul>

<!-- Features -->

### Key Features <a name="key-features"></a>


- **API:** The endpoint generate a random greeting message.
- **REDUX:** Makes the data fetching for the endpoint API and stores the state of the app.
- **REACT:** Displays the greeting message with the greeting component.

### Prerequisites

In order to run this project you need:
- A Terminal.
- A web browser.
- A code editor.
- Ruby,Rails and Postgres Should be installed in your machine.
